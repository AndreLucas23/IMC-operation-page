# 🏃‍♂️ IMC operation page

![Project Cover - Page initial screen screenshot](static/imgs/start.png)

> An elegant, responsive, and reactive Web App for calculating and tracking Body Mass Index (BMI), focused on the best User Experience (UX).

---

## 📌 About the Project

The page has the goal to transform a simple mathematical calculation tool into a pleasant and instant visual experience. Through a clean interface, the user inputs their weight and height data and receives real-time feedback, without needing to reload the page or click confirmation buttons.

---

## ✨ Key Features

* **Real-time Calculation:** The JavaScript reacts to every keystroke (`input event`), instantly updating the dashboard.
* **Dynamic Visual Feedback:** A progress bar changes its width and color (Green, Yellow, Orange, Red) based on the user's current health status.
* **Smart Validation:** The system ignores absurd values (negative, zero, or unrealistic) to ensure calculation integrity.
* **Responsive Design:** Perfectly adapted for mobile, tablet, and desktop screens (Mobile First approach).

## 🛠️ Tech Stack

This project was built purely with core Web technologies, ensuring maximum performance without relying on external libraries:

* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Native variables, Flexbox, state transitions, and responsive design.
* **JavaScript (ES6+):** Object-based architecture, DOM manipulation, and real-time events.

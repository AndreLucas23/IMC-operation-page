# 🏃‍♂️ Página de Cálculo de IMC

![Capa do Projeto - Screenshot da tela inicial da página](static/imgs/start.png)

> Um Web App elegante, responsivo e reativo para cálculo e monitoramento do Índice de Massa Corporal (IMC), focado na melhor Experiência do Usuário (UX).

---

## 📌 Sobre o Projeto

A página tem o objetivo de transformar uma simples ferramenta de cálculo matemático em uma experiência visual agradável e instantânea. Através de uma interface limpa, o usuário insere seus dados de peso e altura e recebe o feedback em tempo real, sem a necessidade de recarregar a página ou clicar em botões de confirmação.

---

## ✨ Funcionalidades Principais

* **Cálculo em Tempo Real:** O JavaScript reage a cada tecla digitada (`input event`), atualizando o painel instantaneamente.
* **Feedback Visual Dinâmico:** Uma barra de progresso altera sua largura e cor (Verde, Amarelo, Laranja, Vermelho) com base no status de saúde atual do usuário.
* **Validação Inteligente:** O sistema ignora valores absurdos (negativos, zerados ou irreais) para garantir a integridade do cálculo.
* **Design Responsivo:** Adaptado perfeitamente para telas de celulares, tablets e desktops (Mobile First).

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído puramente com as tecnologias base da Web, garantindo máxima performance sem dependência de bibliotecas externas:

* **HTML5:** Estrutura semântica e acessibilidade.
* **CSS3:** Variáveis nativas, Flexbox, transições de estado e design responsivo.
* **JavaScript (ES6+):** Arquitetura baseada em objetos, manipulação do DOM e eventos em tempo real.
